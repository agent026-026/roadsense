Roadsense

道路颠簸感知 · iOS · 骑行路况记录（本地传感器与地图）

产品说明

Roadsense（道路颠簸感知）是一款运行在用户 iPhone 上的运动/出行辅助类应用：在骑行过程中，应用使用系统提供的定位（使用 App 期间）记录轨迹并在地图上展示；使用设备的运动传感器（如 userAcceleration）对路面颠簸与冲击做简化检测与分级。每次骑行可保存为一场「会话」，包含路线、距离、冲击次数及历史列表，便于用户回顾。

重要：检测结果受车速、手机固定方式、传感器噪声与 GPS 精度影响，仅供日常参考，不构成专业道路检测结论或安全保证。

功能概要

  • 首次启动欢迎页与使用说明；记录页请求定位与运动权限（用途与系统弹窗文案见 App 内 Info.plist）。
  • 地图展示用户位置（授权后聚焦附近区域）、骑行轨迹_polyline_，以及按路段舒适度着色（绿 / 黄 / 红）。
  • 可调节颠簸检测灵敏度；录制中显示合成加速度约读数（示意）。
  • 会话数据默认以 JSON 形式保存在本机应用沙盒（Application Support），不上传至开发者自有服务器。

注意

建议在真机或支持运动与定位的模拟环境下测试：安装后打开 App → 完成欢迎页 → 在「记录」页允许使用期间定位与运动与健身权限 → 可短距离移动或「开始骑行」观察地图与读数；结束后在「历史」查看会话。若模拟器运动数据不可用，颠簸检测可能受限，属环境限制。

隐私与数据

  • 应用不集成 App 跟踪透明度（ATT）请求；不将数据用于跨 App 广告追踪（与当前隐私清单一致）。
  • 定位与运动数据用于实现声明内的 App 功能；骑行记录默认仅存本机。
  • 用户使用本应用即表示理解传感器与地图功能的风险提示；请遵守交通法规，勿在驾驶或骑行中过度操作手机。

系统要求

  • 需要支持 CoreLocation、CoreMotion 与 MapKit 的设备；部分能力在模拟器上可能不完整。
